# RPA(UiPath) 기반 환율 변동 데이터 메일링 업무 자동화


## 프로젝트 소개
실시간 국제 환율과 수출입 통계를 웹스크래핑하여 이메일로 발송하는 업무 자동화 프로그램 개발

## 개발 기간
 - 2023.10.16 ~ 2023.10.27

### 멤버 구성
 - 김린중(PL/TA): 총괄
 - 박서우(AA): 서비스 기획, PPT 제작 및 발표
 - 전유진(TA): 전체 시퀀스 구현

### 개발 환경
 - 운영 체제: Windows 10
 - 개발 도구: UiPath, Excel, Outlook
